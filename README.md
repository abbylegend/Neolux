# keycloudin (AirplaneDev OS v3.5)

keycloudin is a landing-page hosted coding cloud for beginners with full-stack training tools.

## Included capabilities
- Project generator, editor, terminal, analyzer, preview, and stats dashboard.
- AI chat activation with **Key** (lingual coding agent).
- Agent catalog with specialized agents:
  - Nora (Ruby)
  - Sora (HTML)
  - Tiki (Python)
  - Volt (JavaScript)
  - Forge (Java)
  - Zucchini (ethical hacking tutor with 3-day trial + monthly/annual pricing)
- User account registration/login and profile creation.
- Admin-only database snapshot viewer.
- Automated messaging broadcast for huge programs.
- Download active files and full project folder export to local storage.

## Run
```bash
npm install
npm start
```
Open `http://localhost:3000`.
